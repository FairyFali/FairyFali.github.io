и§ЈеҶізҡ„дё»иҰҒй—®йўҳ

  • ж №жҚ®ж–ҮжЎЈзҡ„еҶ…йғЁиҜҚзҡ„еҲҶеёғжғ…еҶөпјҢжқҘжӢҹеҗҲдёӨзұ»еҲҶеёғdoc-topгҖҒtop-wordпјҢиғҪеӨҹеә”з”ЁеҲ°ж–Үжң¬жЁЎеһӢгҖҒеҚҸеҗҢиҝҮж»Өе’Ңж–Үжң¬еҲҶзұ»дёӯгҖӮ

жҸҗеҮәзҡ„дё»иҰҒж–№жі•е’ҢиҙЎзҢ®

  • ж–ҮжЎЈйӣҶDдҪңдёәиҫ“е…ҘпјҢи®ӯз»ғдёӨзұ»еҲҶеёғdoc-topе’Ңtop-wordпјҢ
    • жҜҸдёҖдёӘdocеҜ№еә”дёҖдёӘtopзҡ„еӨҡйЎ№ејҸеҲҶеёғ$\theta_d(p_{t1},вҖҰ,p_{tk})$пјҢ$ti$иЎЁзӨәз¬¬$i$дёӘдё»йўҳпјҢ$p_{ti}=\frac{n_{ti}}{n}$;
    • еҗҢзҗҶеҜ№дәҺжҜҸдёҖдёӘдё»йўҳtopеҜ№еә”дёҖдёӘwordзҡ„еӨҡйЎ№ејҸеҲҶеёғ$\varphi_t(p_{w1},вҖҰ,p_{wm})$.
  • и®ӯз»ғиҝҮзЁӢпјҡ
    1. йҡҸжңәеҲқе§ӢеҢ–$\theta,\varphi$пјҢжҢүз…§е…ҲйӘҢжҰӮзҺҮp(d)йҖүжӢ©дёҖзҜҮж–ҮжЎЈ$d_i$пјӣ
    2. д»ҺDirichletеҲҶеёғ$\alpha$дёӯеҸ–ж ·з”ҹжҲҗж–ҮжЎЈ$d_i$зҡ„дё»йўҳеҲҶеёғ$\theta_i$пјӣ
    3. д»Һдё»йўҳзҡ„еӨҡйЎ№ејҸеҲҶеёғ$\theta_i$дёӯеҸ–ж ·з”ҹжҲҗж–ҮжЎЈ$d_i$зҡ„з¬¬$j$дёӘиҜҚдё»йўҳ$z_{i,j}$пјӣ
    4. д»ҺDirichletеҲҶеёғ$\beta$дёӯеҸ–ж ·з”ҹжҲҗдё»йўҳ$z_{i,j}$еҜ№еә”зҡ„иҜҚиҜӯеҲҶеёғ$\varphi_{z_{ij}}$пјӣ
    5. д»ҺиҜҚиҜӯеӨҡйЎ№ејҸеҲҶеёғдёӯз”ҹжҲҗж–°иҜҚ$w_{i,j}$.
  • дјҳеҢ–иҝҮзЁӢ

  • жҷ®йҖҡзҡ„KNNж–№жі•ж— жі•еӨ„зҗҶзҰ»зҫӨзӮ№е’ҢеҜҶеәҰзҡ„еҪұе“ҚпјҢжң¬ж–ҮжҸҗеҮәзҡ„ејәKNNж–№жі•еҸҜд»ҘеңЁзәҝжҖ§ж—¶й—ҙеӨҚжқӮеәҰеҶ…и§ЈеҶіиҝҷдёӘй—®йўҳе№¶дё”дёҺзҺ°жңүзҡ„еҮ з§Қж–№жі•д»ҘеҸҠSVMгҖҒLRж–№жі•иҝӣиЎҢиҜ•йӘҢеҲҶжһҗеҫ—еҲ°зҡ„жңҖдјҳзҡ„з»“жһңгҖӮ

жҸҗдҫӣзҡ„дё»иҰҒж–№жі•е’ҢиҙЎзҢ®

  • е°Ҷж•°жҚ®жҠҪиұЎдёәдёҖдёӘж— еҗ‘еӣҫпјҢиҠӮзӮ№жҳҜж•°жҚ®пјҢиҫ№жҳҜж•°жҚ®д№Ӣй—ҙзҡ„и·қзҰ»пјӣ
  • и§ЈеҶізҡ„дёӨдёӘй—®йўҳпјҡ
  • KNNдёӨжӯҘејӮеёёзӮ№жЈҖжөӢз®—жі•пјҡиҫ“е…Ҙ$G^+(O_{train} + v,D_{train}+d(v, O_{train}))$пјҢе’ҢжөӢиҜ•ж•°жҚ®vпјҢзҰ»зҫӨйҳҲеҖјTпјӣиҫ“еҮәvжҳҜеҗҰдёәзҰ»зҫӨзӮ№е’Ңvзҡ„KдёӘйӮ»еұ…гҖӮйҒҚеҺҶжҜҸдёӘи®ӯз»ғж•°жҚ®пјҢи®Ўз®—жҜҸдёӘж•°жҚ®зҡ„з¬¬KдёӘйӮ»еұ…зҡ„и·қзҰ»$Ndist^+_{v,k}$пјҢеҰӮжһңеӨ§дәҺзӯүдәҺ$D_{iv}$пјҢеҲҷеҠ е…ҘM(v)пјҢйҒҚеҺҶз»“жқҹеҗҺеҲӨж–ӯеҰӮжһң$len(M(v))<T$еҲҷеҲӨе®ҡдёәејӮеёё,йӮ»еұ…иҠӮзӮ№з”ұ$Ndist^+_{v,k}$еҶіе®ҡгҖӮ
  • дёҠдёӢж–Үж•Ҹж„ҹзҡ„ејәKNNз®—жі•пјҡдё»иҰҒдёәдәҶи§ЈеҶідёҚеҗҢж•°жҚ®йӣҶеҜҶеәҰдёҚеҗҢзҡ„й—®йўҳпјҢеҸҜжҸҗдҫӣжЈҖжөӢејӮеёёгҖӮиҫ“е…Ҙеӣҫ$G^+$е’ҢжөӢиҜ•ж•°жҚ®vпјӣиҫ“еҮәvжҳҜеҗҰдёәејӮеёёзӮ№е’Ңе°‘дәҺKдёӘдёҙиҝ‘зӮ№пјӣйҒҚеҺҶжөӢиҜ•ж•°жҚ®пјҢи®Ўз®—жҜҸдёӘж•°жҚ®зҡ„з¬¬kдёӘйӮ»еұ…зҡ„и·қзҰ»пјҢеҰӮжһңж•°жҚ®iеңЁvзҡ„kдёӘйӮ»еұ…еҶ…дё”vеҲ°ж•°жҚ®iзҡ„и·қзҰ»е°ҸдәҺiеҲ°з¬¬kдёӘйӮ»еұ…зҡ„и·қзҰ»еҲҷе°Ҷiж”ҫе…ҘN(v)дёӯпјҢз»“жқҹйҒҚеҺҶпјҢеҰӮжһңN(v)дёәз©әеҲҷиҜҙжҳҺvжҳҜејӮеёёзӮ№пјҢиҝ”еӣһN(v)дёәvзҡ„йӮ»еұ…зӮ№гҖӮ
  • й«ҳж•Ҳзҡ„ејәKNNз®—жі•пјҡеңЁдёҠдёҖдёӘз®—жі•зҡ„еҹәзЎҖдёҠеҜ№ж—¶й—ҙеӨҚжқӮеәҰиҝӣиЎҢж”№иҝӣпјҢйҖӮеә”дәҺжөӢиҜ•ж•°жҚ®еҸҳеҢ–зҡ„жғ…еҶөгҖӮиҫ“е…ҘGе’ҢжөӢиҜ•йӣҶеҗҲV={v,вҖҰ};иҫ“еҮәжөӢиҜ•ж•°жҚ®зҡ„жҳҜеҗҰдёәејӮеёёзӮ№д»ҘеҸҠйӮ»еұ…иҠӮзӮ№гҖӮи®ӯз»ғйҳ¶ж®өпјҢи®Ўз®—еҮәGдёӯжүҖжңүиҠӮзӮ№зҡ„$Ndist_{i,K-1}$е’Ң$Ndist_{i,k}$пјҢж—¶й—ҙеӨҚжқӮеәҰдёә$O(K^2N)$пјҢжөӢиҜ•йҳ¶ж®өпјҢйҒҚеҺҶжҜҸдёҖдёӘжөӢиҜ•ж•°жҚ®vпјҢеҫ—еҲ°ж–°зҡ„еӣҫ$G^+$пјҢеҜ№дәҺжҜҸдёҖдёӘжөӢиҜ•ж•°жҚ®iпјҢеҰӮжһң$Ndist_{i,K-1} == Ndist_{i,K}$пјҢеҲҷ$Ndist^+_{i,K} = Ndist_{i,K}$пјҢеҗҰеҲҷ$Ndist^+_{i,K} = min(Ndist_{i,K-1}, D^+_{iv})$пјҢеҶҚи®Ўз®—vзҡ„з¬¬kдёӘйӮ»еұ…зҡ„и·қзҰ»$Ndist^+_{v,K}$ж—¶й—ҙеӨҚжқӮеәҰдёә$O(KN)$жңҖеҗҺе’ҢдёҠдёҖз®—жі•еҲӨж–ӯжҳҜеҗҰдёәејӮеёёзӮ№зҡ„жқЎд»¶зӣёеҗҢгҖӮ
